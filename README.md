# Selenium Automation Framework

A Selenium WebDriver automation project built using Java and Maven.

This repository contains automated browser test scripts created as part of Selenium learning projects, likely based on Udemy Selenium course examples. 
The framework demonstrates how to automate web application testing using Selenium WebDriver in Java.
